# RANGO FÁCIL
Este é um projeto desenvolvido na disciplina Projeto e Prática no curso de Informática para Internet do IFPE – Campus Igarassu.

## Visão Geral
O Rango Fácil é um projeto desenvolvido como um site/aplicativo voltado para a comercialização de lanches no Campus Igarassu do IFPE.

## Objetivo Principal
A plataforma tem como objetivo facilitar a interação entre vendedores e clientes, oferecendo um ambiente digital organizado para a divulgação de produtos e a realização de pedidos.

## Objetivo Principal
O objetivo principal do sistema é otimizar o processo de venda e compra de lanches, permitindo que clientes realizem pedidos de forma prática e que vendedores gerenciem seus produtos e pedidos de maneira eficiente.

## Público-Alvo
O sistema é destinado a estudantes e funcionários do Campus Igarassu do IFPE, que podem se cadastrar como clientes ou vendedores.

## Funcionalidades do Sistema
- Funcionalidades para Clientes
- Realizar cadastro e login no sistema;
- Visualizar a lista de vendedores e lanches disponíveis;
- Selecionar lanches para compra;
- Confirmar pedidos;
- Acompanhar o status dos pedidos.
- Funcionalidades para Vendedores
- Realizar cadastro e login no sistema;
- Cadastrar produtos com nome, descrição e preço;
- Gerenciar pedidos realizados pelos clientes;
- Listagem de lanches;
- Área de pedidos do cliente;
- Painel do vendedor.
- Considerações Finais

## Tecnologias Utilizadas
- Laravel — framework backend
- Vue.js — framework frontend
